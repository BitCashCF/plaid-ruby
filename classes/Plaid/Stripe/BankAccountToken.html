Creates a Stripe bank account token from an access_token.

Does a POST /processor/stripe/bank_account_token/create call which can be used to generate a Stripe bank account token for a given account ID.

access_token

access_token to create a public token for

account_id

ID of the account to create a bank account token for

Returns

Returns CreateResponse object containing a Stripe bank account token.

def create(access_token, account_id)
  post_with_auth 'processor/stripe/bank_account_token/create',
                 CreateResponse,
                 access_token: access_token, account_id: account_id
end
