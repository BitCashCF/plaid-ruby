Class Plaid::Models::AssetReport < Plaid::Models::BaseModel

  • lib/plaid/models.rb

A representation of an asset report.

Attributes
[R] asset_report_id

Plaid's unique identifier for this asset report.
[R] client_report_id

An identifier you determine and submit for the Asset Report.
[R] date_generated

The date and time when the Asset Report was created.
[R] days_requested

The duration of transaction history you requested.
[R] items

Data returned by Plaid about each of the Items included in the Asset Report; see AssetReportItem object for fields.
[R] user

Data submitted by you about the user whose Asset Report is being compiled; see AssetReportUser object for fields.
