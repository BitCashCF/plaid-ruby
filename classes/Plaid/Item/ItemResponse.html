Class Plaid::Item::ItemResponse < Plaid::Models::BaseResponse

  • lib/plaid/products/item.rb

Response for /item/create and /item/mfa endpoints.

Attributes
[R] access_token

The String access_token to use with API.
[R] device

The MFA/OTP device information: Plaid::Models::MFA::Device.
[R] device_list

The list of devices to send the OTP to: Array of Plaid::Models::MFA::DeviceListElement.
[R] item

The Plaid::Models::Item object, returned if item has been successfully created.
[R] mfa_type

The String MFA type. E.g. “device_list”, “device”, “questions”, “selections”.
[R] questions

The Array of String MFA questions.
[R] selections

The Array of MFA selections: Plaid::Models::MFA::Selection.
